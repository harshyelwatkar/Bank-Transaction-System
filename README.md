# Bank Transaction System

## Overview

The Bank Transaction System is a RESTful backend application designed to manage users, accounts, and banking transactions. It provides secure transaction processing and reliable API operations.

## Features

- User and account management.
- Deposit and withdrawal transactions.
- Account balance updates.
- MongoDB-based data persistence.
- Validation and error handling.
- Modular REST API architecture.

## Tech Stack

- Backend: Node.js
- Framework: Express.js
- Database: MongoDB
- API Testing: Postman
- Version Control: GitHub

## Installation & Setup

### Clone the Repository

```bash
git clone <repository-url>
cd bank-transaction-system
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file and configure MongoDB connection details.

### Start the Server

```bash
npm start
```
